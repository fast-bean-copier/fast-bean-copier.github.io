欢迎使用 Fast Bean Copier

Fast Bean Copier 是一个基于注解处理器的 Java Bean 拷贝工具，用于生成类型安全、高性能且零依赖的 Bean 映射代码。

v1.3.1 更新：统一所有集合类型的 UnaryOperator 行为（List/Set/Map/Array）、Properties 配置文件支持、逆向转换智能跳过特殊字段。

✨ 核心特性

⚡ 高性能

编译期代码生成，零运行时反射，性能与手写代码相同

🔒 类型安全

编译期类型检查，避免运行时错误

📦 零依赖

生成的代码不依赖任何外部库

🎯 简洁易用

只需一个 @CopyTarget 注解，自动生成所有拷贝方法

🔄 多字段映射

支持多对一、一对多转换，灵活的字段合并与拆分

� 类型文转换器

6 种内置 TypeConverter，支持数字/日期格式化、枚举转换、JSON 序列化

💉 依赖注入

支持 Spring、CDI、JSR-330 框架集成，构造器注入

📝 表达式映射

支持 Java 表达式进行复杂字段转换

🎨 函数式定制

UnaryOperator 重载，统一的集合级别定制（List/Set/Map/Array），支持过滤、排序、不可变转换（v1.3.1 统一）

� 更新现有对象

updateDto/updateEntity 方法，更新已存在的对象而非创建新对象（v1.3 新增）

⚙️ 映射前回调

beforeMapping 属性，在映射前执行验证、初始化等操作（v1.3 新增）

🎯 条件映射

condition 属性，根据条件决定是否执行字段映射（v1.3 新增）

📋 默认值和常量

defaultValue/constant 属性，设置默认值或常量值（v1.3 新增）

🌐 全局配置

@CopyTargetConfig 包级别配置 + Properties 配置文件支持，统一管理组件模型和 Null 值策略（v1.3/v1.3.1）

🚀 快速开始

只需三步即可开始使用 Fast Bean Copier：

1. 添加依赖

<dependency>
    <groupId>com.github.jackieonway</groupId>
    <artifactId>fast-bean-copier-annotations</artifactId>
    <version>1.3.1</version>
</dependency>

<dependency>
    <groupId>com.github.jackieonway</groupId>
    <artifactId>fast-bean-copier-processor</artifactId>
    <version>1.3.1</version>
    <scope>provided</scope>
</dependency>

2. 定义 DTO 类

import com.github.jackieonway.copier.annotation.CopyTarget;

@CopyTarget(source = User.class)
public class UserDto {
    private Long id;
    private String name;
    private String email;
    // getter/setter...
}

3. 使用

User user = new User(1L, "张三", "zhangsan@example.com");
UserDto userDto = UserDtoCopier.toDto(user);
