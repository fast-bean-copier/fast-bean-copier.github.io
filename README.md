# Fast Bean Copier 文档网站

这是 Fast Bean Copier 1.0.0 的官方文档网站。

## 文档页面

- **首页** (`index.html`) - 项目介绍和快速导航
- **快速入门** (`getting-started.html`) - 5 分钟快速开始指南
- **参考文档** (`reference.html`) - 完整的功能参考文档
- **API 文档** (`api.html`) - 详细的 API 参考
- **常见问题** (`faq.html`) - 常见问题和解决方案
- **项目总结** (`summary.html`) - 项目概览和统计信息

## 本地查看

在浏览器中打开 `index.html` 即可查看文档。
